# test - FFire Python Bindings

Python bindings for the test schema, generated by [FFire](https://github.com/shaban/ffire).

## Installation

### Recommended: Virtual Environment

```bash
cd python
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install .
```

### Alternative: User Installation

If you don't want to use a virtual environment:

```bash
pip install --user .
```

### System Python Warning

If you see a warning about "externally managed environment" (PEP 668), this is normal on some systems (Homebrew Python, modern Linux). Use one of the methods above - the virtual environment approach is recommended.

## Usage

```python
from test import Message

# Decode from binary
with open('data.bin', 'rb') as f:
    data = f.read()

msg = Message.decode(data)

# Encode back to binary
encoded = msg.encode()
```

## API

### `Message`

**`Message.decode(data: bytes) -> Message`**

Decode a `Message` from binary data.

- **Parameters:** `data` - Binary data (bytes)
- **Returns:** `Message` object
- **Raises:** `RuntimeError` if decoding fails

**`Message.encode() -> bytes`**

Encode this `Message` to binary data.

- **Returns:** Binary data (bytes)
- **Raises:** `RuntimeError` if encoding fails

## Platform Support

This package includes pre-compiled libraries for:

- macOS (Darwin): `libffire.dylib`
- Linux: `libffire.so`
- Windows: `ffire.dll`

The correct library is automatically loaded based on your platform.

## Requirements

- Python 3.7 or higher
- ctypes (included in Python standard library)
